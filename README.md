Step 1 Install dependencies
pip install dilithium-py qrcode

cd Freedom-coin

Step 3 Create your wallet
python3 wallet.py

Step 4 Start mining
python3 my_miner.py

Step 5 Use master wallet
python3 freedom_master.py
