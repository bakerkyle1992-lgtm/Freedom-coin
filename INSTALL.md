# FREEDOM Coin Installation Guide

## Requirements
- Python 3.8 or higher
- Android phone with Termux or any Linux computer

## Quick Install

Step 1 - Install Termux from F-Droid on Android

Step 2 - Setup
Run: pkg update and pkg upgrade
Run: pkg install python git
Run: pip install dilithium-py qrcode

Step 3 - Download FREEDOM Coin
Run: git clone https://github.com/bakerkyle1992-lgtm/Freedom-coin
Run: cd Freedom-coin

Step 4 - Create Your Wallet
Run: python3 wallet.py

Step 5 - Start Mining
Run: python3 my_miner.py

Step 6 - Use Master Wallet
Run: python3 freedom_master.py

## Security
- Write down your 12 word recovery phrase
- Never share your private key
- Use emergency freeze if phone stolen
- Set up family shield for large amounts
